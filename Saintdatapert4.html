def calculate_statistics(audio_path):
    y, sr = librosa.load(audio_path)

    # UNTUK MENGHITUNG NILAI STATISTIKA
    mean = np.mean(y)
    std_dev = np.std(y)
    max_value = np.max(y)
    min_value = np.min(y)
    median = np.median(y)
    skewness = skew(y)  # Calculate skewness
    kurt = kurtosis(y)  # Calculate kurtosis
    q1 = np.percentile(y, 25)
    q3 = np.percentile(y, 75)
    mode_value, _ = mode(y)  # Calculate mode
    iqr = q3 - q1

    # UNTUK MENGHITUNG NILAI ZCR
    zcr_mean = np.mean(librosa.feature.zero_crossing_rate(y=y))
    zcr_median = np.median(librosa.feature.zero_crossing_rate(y=y))
    zcr_std_dev = np.std(librosa.feature.zero_crossing_rate(y=y))
    zcr_kurtosis = kurtosis(librosa.feature.zero_crossing_rate(y=y)[0])
    zcr_skew = skew(librosa.feature.zero_crossing_rate(y=y)[0])

    # UNTUK MENGHITUNG NILAI RMSE
    rmse = np.sum(y**2) / len(y)
    rmse_median = np.median(y**2)
    rmse_std_dev = np.std(y**2)
    rmse_kurtosis = kurtosis(y**2)
    rmse_skew = skew(y**2)

    return [zcr_mean, zcr_median, zcr_std_dev, zcr_kurtosis, zcr_skew, rmse, rmse_median, rmse_std_dev, rmse_kurtosis, rmse_skew]

Modelling#

KNeighborClasifier (KNN)

K-Nearest Neighbors (KNN) adalah algoritma machine learning yang digunakan untuk tugas klasifikasi dan regresi. Algoritma ini termasuk dalam kategori algoritma berbasis instan, yang berarti modelnya mempelajari pola langsung dari data pelatihan yang disimpan. KNN mengandalkan perbandingan kemiripan antara data baru dan data pelatihan untuk membuat prediksi.

  • K: K adalah parameter dalam KNN yang menentukan jumlah tetangga terdekat yang akan dipertimbangkan saat membuat prediksi. Misalnya, jika K = 3, model akan mempertimbangkan tiga tetangga terdekat untuk menentukan kelas atau nilai regresi.

  • Fungsi Jarak: KNN menggunakan fungsi jarak (misalnya, Euclidean distance atau Manhattan distance) untuk mengukur seberapa dekat atau jauhnya suatu data baru dari data pelatihan. Jarak ini digunakan untuk mengidentifikasi tetangga terdekat.

  • Voting untuk Klasifikasi: Pada tugas klasifikasi, KNN melakukan â€œvotingâ€ di antara tetangga terdekat untuk menentukan kelas mayoritas. Misalnya, jika K = 3 dan dua tetangga termasuk ke dalam kelas A dan satu ke dalam kelas B, data baru akan diklasifikasikan sebagai kelas A.

  • Rata-rata untuk Regresi: Pada tugas regresi, KNN mengambil rata-rata nilai target dari tetangga terdekat untuk menghasilkan prediksi nilai kontinu.

  • Skala Fitur: KNN sangat sensitif terhadap skala fitur. Penting untuk memastikan bahwa semua fitur memiliki skala yang serupa agar pengaruhnya terhadap perhitungan jarak seimbang.

  • Tuning Parameter K: Pemilihan nilai K yang optimal dapat mempengaruhi kinerja model. Nilai K yang terlalu kecil dapat menyebabkan model menjadi sensitif terhadap noise, sementara nilai K yang terlalu besar dapat menyebabkan model menjadi terlalu umum

#Rumus K-NN \(d(X_{\text{new}}, X_i) =\sqrt{\sum_{j=1}^{n}(X_{\text{new}, j} - X_{i, j})^2}'))\)

#Penggunaan Rumus KNN

  • Menentukan Parameter K: Pilih jumlah tetangga terdekat (K) yang akan digunakan dalam proses klasifikasi atau regresi. Nilai K ini sangat mempengaruhi kinerja algoritma.

  • Menghitung Jarak: Gunakan suatu metrik jarak (seperti Euclidean, Manhattan, Minkowski, dll.) untuk mengukur jarak antara titik data yang ingin diprediksi dan setiap titik data lainnya dalam dataset.

  • Mengidentifikasi Tetangga Terdekat: Temukan K tetangga terdekat berdasarkan jarak yang dihitung sebelumnya. Ini adalah tetangga-tetangga yang memiliki jarak terdekat dengan data yang ingin diprediksi.

  • Voting (Klasifikasi) atau Averaging (Regresi): Klasifikasi: Dalam klasifikasi, atribut target data yang ingin diprediksi akan ditentukan oleh mayoritas kelas dari tetangga-tetangga terdekat. Misalnya, jika sebagian besar tetangga adalah kelas A, maka data yang ingin diprediksi juga diklasifikasikan sebagai kelas A. Regresi: Dalam regresi, atribut target data yang ingin diprediksi akan diestimasi dengan mengambil rata-rata nilai dari atribut target tetangga-tetangga terdekat.

  • Prediksi: Gunakan hasil dari langkah sebelumnya sebagai prediksi untuk data yang ingin diprediksi.

  • Pentingnya Parameter K: Nilai K yang terlalu kecil dapat membuat model sensitif terhadap noise atau outlier. Nilai K yang terlalu besar dapat menyebabkan model kurang fleksibel dan tidak dapat menangkap pola yang kompleks dalam data

import numpy as np
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score

# Menentukan jumlah K (jumlah tetangga terdekat yang akan diuji)
K = 30

# Inisialisasi array untuk menyimpan akurasi untuk setiap nilai K
acc = np.zeros((K-1))

# Melakukan iterasi untuk nilai K dari 1 hingga K dengan loncatan 2 (karena K selalu harus ganjil untuk menghindari keputusan seimbang)
for n in range(1, K, 2):
    # Membuat model KNN dengan nilai K saat ini dan metrik jarak Euclidean
    knn = KNeighborsClassifier(n_neighbors=n, metric="euclidean")

    # Melatih model dengan data latih yang telah dinormalisasi
    knn.fit(X_train_scaled, y_train)

    # Melakukan prediksi pada data uji
    y_pred = knn.predict(X_test_scaled)

    # Menghitung akurasi dan menyimpannya dalam array
    acc[n-1] = accuracy_score(y_test, y_pred)

# Menampilkan nilai K dengan akurasi terbaik
print('Akurasi terbaik adalah', acc.max(), 'dengan nilai K =', acc.argmax()+1)
